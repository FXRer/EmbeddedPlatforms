The μracoli IEEE-802.15.4-Sniffer


Abstract

With a Sensor Terminal Board or a STK541 combined with a Radio Controller Board, a version of wireshark and the μracoli-wireshark bridge script you can quickly setup a 802.15.4 sniffer and watch the traffic on your wireless sensor network. This application package illustrates the steps how this open source sniffer solution is installed.
