This zip contains a code example of how communicate with a Cirrus
CS8900A Ethernet controller using a conventional 8051 processor
8-bit bus.  The 8051 polls the CS8900A to see if a message has
arrived. 

Note that the CS8900 does not support interrupts in 8-bit mode
mode.  Chip connections are per Cirrus app note AN181.


Refererences (www.cirrus.com)

  AN181  Using CS8900 in 8-bit mode
  AN205  CS8900 FAQ
